# BIO-TRACK

> 简洁的减脂热量与宏量营养素计算工具（纯前端）

## ✨ 项目简介

BIO-TRACK 是一个基于浏览器的减脂计算工具，提供 **BMR / TDEE / 目标摄入** 的快速估算，并根据体重自动给出 **蛋白质 / 脂肪 / 碳水** 分配。
支持本地档案管理与完整设置系统，无需后端即可使用。

## 🚀 核心功能

### 🔢 计算模块

* BMR（Mifflin-St Jeor 公式）
* TDEE（按活动系数计算）
* 目标摄入量（热量缺口）
* 三大营养素自动分配（P/F/C）

### 📊 可视化

* 目标热量大屏展示
* 宏量营养素进度条
* kcal 与占比显示切换

### 📂 档案系统

* 多档案保存（LocalStorage）
* 查看 / 载入 / 删除
* 显示完整计算结果（BMR / TDEE / 摄入 / 宏量）

### ⚙️ 设置系统

* 主题：深色 / 浅色 / 跟随系统
* 主色调：蓝 / 绿 / 紫 / 橙
* 字体大小 / 紧凑布局
* 显示控制（小数位 / kcal / 百分比）
* 默认计算参数
* 数据导出 / 导入 / 清空

## 💾 数据说明

* 所有数据存储在浏览器 LocalStorage
* 不会上传服务器
* 清除浏览器数据会导致档案丢失
* 不同浏览器之间不共享数据
